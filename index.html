Styles CSS:

  1. Width: ширина нашего элемента
  2. Height: высота нашего элемента
  3. text-decoration: украшение для текста
  4. font-size:размер текста
  5. font-family: шрифт текста
  6. font-weight: толшина текста
  7. color: цвет текста
  8. background-color: цвет заднего фона нашего элемента
  9. list-style: стиль нашего li элемента

HTML new Elem

  1. a - элемент для ссылки
  2. img - элемент для загрузки фото
  3. class atribut - атрибут внутри элемента и выбирается в виде "."
  4. id atribut - атрибут внутри элемента и выбирается в виде "#"
